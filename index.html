Розглянутий список команд для роботи з терміналом

  • Git init - створення респозіторія
  • Git status - перевірка поточного стану респозіторія
  • Git add - додає зміни робочего каталогу в індекс
  • Git commit - зберігає додані зміни
  • Git reset - відміняє додані зміни до останнього commit
